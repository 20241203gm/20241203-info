## 개요

이 프로젝트는 2024년 12월 3일 발생한 계엄령과 그 이후의 정치적 사건들을 기록하고, 대한민국 민주주의의 회복 과정을 담아내는 것을 목표로 합니다.

## 기술 스택

- Frontend: Next.js, GSAP, D3.js
- Database: Supabase
- Hosting: Vercel
- Media Storage: Cloudinary

## 개발 환경 설정

1. 저장소 클론
```bash
git clone https://github.com/your-username/20241203-info.git
cd 20241203-info
```

2. 의존성 설치
```bash
npm install
```

3. 개발 서버 실행
```bash
npm run dev
```
